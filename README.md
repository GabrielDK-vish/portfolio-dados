# 📊 Gabriel R. - Portfolio de Dados

Portfolio desenvolvido em Python utilizando Streamlit com foco em análise de dados, visualização e storytelling analítico voltado para negócios.

O projeto possui painéis interativos simulando cenários reais de:
- E-commerce
- Marketing

---

# 🚀 Tecnologias Utilizadas

- Python
- Streamlit
- Pandas
- Plotly
- SQLAlchemy
- SQLite
- Scikit-Learn
- Matplotlib
- Seaborn

# ⚙️ Como Executar o Projeto

## 1. Clone o repositório

```bash
git clone URL_DO_REPOSITORIO
```

---

## 2. Acesse a pasta do projeto

```bash
cd portfolio_dados
```

---

## 3. Crie a Virtual Environment

```bash
python -m venv venv
```

---

## 4. Ative a Virtual Environment

### Windows CMD

```bash
venv\Scripts\activate
```

### PowerShell

```powershell
.\venv\Scripts\Activate.ps1
```

---

## 5. Instale as dependências

```bash
pip install -r requirements.txt
```

---

## 6. Execute o projeto

```bash
streamlit run app.py
```
